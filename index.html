Visual Studio Code (VSCode) is a cross-platform (Linux, Mac OS, Windows) editor that can be extended with plugins to your needs. For example, if you want to create an ASP.NET application using Visual Studio Code you need to perform several steps on your own to setup the project.


Console.log()

The console. log() is a function that writes a message to log on the debugging console, such as Webkit or Firebug. In a browser you will not see anything on the screen. It logs a message to a debugging console. It is only available in Firefox with Firebug and in Webkit based browsers (Chrome and Safari).

Note:The actual digits logged in the console are (25 , 5)
